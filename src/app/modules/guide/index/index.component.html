Firebase Hosting

Firebase Hosting - высококласный хостинг размещения контента, предоставляет быстрый и безопасный способ размещения статичесих и динамических сайтов. При помощи одной команды можно быстро развернуться приложением, используя собственную распределенную сеть доставки контента (CDN).

Ка это работает

Firebase Hosting создан для современных разработчиков. Вебсайты и приложения стали более мощными с применением JavaScript фреймворков таких как Angular или инструментов статической генерации, как Jekyll. Вне зависимости от сложности развертываемого приложения, хостинг предлагает современную инфраструктуру, функции, и инструментарий связывая вместе разработку и управление развернутыми вебсайтами и приложениями.
Хостинг предлагает поддомен на домене firebaseapp.com. Используя инструмент Firebase CLI можно развернуться приложением из указанной локальной директории на компьютере разработчика на сервер хостинга. Расширяя границы статического контента можно использовать функциональность Cloude Functions (облачные функции) для раздачи динамического контента приложения. Весь контент раздается по безопасному протоколу, через SSL соединение, используя ближайший к пользователю сервер распределенной, глобальной сети доставки контента.
В дополнение к размещению контента, Firebase Hosting предлагает очень простые настройки конфигурации для построения сложных прогрессивных веб приложений PWA, предлагая быстрое переопределение клиентской маршрутизации или настройки сетевых заголовков. Как только, разработчик готов выйти в продакшн на стороннем домене, он может развернуть его на Firebase Hosting используя механизм развертывания сторонних доменов, и хостин предлагает автоматическое предоставление сертификата SSL для безопасной работы домена.
